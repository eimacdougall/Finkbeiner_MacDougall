# Finkbeiner_MacDougall

# Install Necessary Packages
To install the packages run 'pip install -r requirements.txt' in the terminal

# Starting MLFlow Run
In the terminal run 'mlflow ui' and click the hosting link at the top of the output "http://127.0.0.1:5000"
If your port does not match this, then you may need to run 'mlflow ui --port 5000'
